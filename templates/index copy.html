Neon Fire

Git
Game Development

Neon fire is a bullet-hell rogue-like game made with the Godot Engine. This project was made for my game development class. A lot of game I enjoy are rogue-like games, so I decided to make one myself using an engine I was familiar with.

While this game isn't fully fleshed out the way I envisioned it (due to scope creep, which I've now learned not to do). A lot of the important systems are already in place, and all thats needed is content. Systems including Random level generation with a zelda-like lock and key progression system to get through the floor. Unique item rooms and key rooms are guarded by waves of enemies which drop consumables when killed. All culminating into a final boss battle, that you see in the preview video.

Yapama (Yet Another Password Manager)

Git
Mobile Development

This is a school project for my Mobile Development class where we were tasked to make any sort of Mobile App. I decided to make a password manager that would give the user more options in storing their password databases.

Rather than storing your passwords on app creators servers like most password managers do. Yapama would allow the users to store their passwords using Drive, Git, or FTP servers, give the users peace of mind when it comes to the security of their passwords.
